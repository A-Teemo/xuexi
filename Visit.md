## 在同一局域网内访问其他电脑
```
sudo apt install openssh-server    (安装本地可以访问其他IP)
比如复制到其他IP上 :  scp a.html(源)  (电脑用户名)linux@10.198.1.214:~/        这样就复制到他的用户目录下了
ssh linux@10.198.1.214 可以进入其他电脑
------------------------------------------------------------------------------------
进入/etc/vim/vimrc  在最后面可以加入 set nu 就可以有数字的列表
```
