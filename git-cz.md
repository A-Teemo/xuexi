##　git基本操作
```
git init (初始化)
git status(查看)
git clone http://
git add .(全部) or 文件名    ----(关注)
git commit  (提交)
git pull (从git中更新到本地)
git push origin master (从本地提交到git)
git diff (比较提交前后跟现在版本的差异)
vim gifignre (加入需要忽略的文件)
git format-patch  .. 引出文件
git am ../ 引入文件
git branch 文件名     (创建分支)
git branch -av    (查分支)
git checkout 文件名 (切换分支)
git merge 文件名 (合并分支)
git reset --hard id  (返回版本)

```
